# SVISH

A lightweight, hackable, shell that's a work in progress by the PNDTC for shiggles.

SVISH probably only works on UNIX-based systems.

SVISH stands for SVI SHell, SVI stands for something, but nobody is really sure what.

# Build Instructions

Note, go 1.4+ is required to build svish.

Linux: Run LinuxBuild.sh

Windows: Run WindowsBuild.bat

Mac OS X: Run MacBuild.sh
